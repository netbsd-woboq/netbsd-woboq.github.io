439/* Overview of LBER tag construction
440 *
441 * Bits
442 * ______
443 * 8 7 | CLASS
444 * 0 0 = UNIVERSAL
445 * 0 1 = APPLICATION
446 * 1 0 = CONTEXT-SPECIFIC
447 * 1 1 = PRIVATE
448 * _____
449 * | 6 | DATA-TYPE
450 * 0 = PRIMITIVE
