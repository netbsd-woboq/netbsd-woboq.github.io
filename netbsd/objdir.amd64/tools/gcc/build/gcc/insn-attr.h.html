218/* The following function returns negative value if given
219 insn can be issued in processor state described by given
220 DFA state. In this case, the DFA state is changed to
221 reflect the current and future reservations by given
222 insn. Otherwise the function returns minimal time
223 delay to issue the insn. This delay may be zero
224 for superscalar or VLIW processors. If the second
225 parameter is NULL the function changes given DFA state
226 as new processor cycle started. */
227extern int state_transition (state_t, rtx);
228

234 macro AUTOMATON_STATE_ALTS is defined because its
235 implementation may require much memory. */
236extern int state_alts (state_t, rtx);
