32/*
33 * Create a snapshot of the file system currently mounted on the first argument
34 * using the second argument as backing store and return an open file
35 * descriptor for the snapshot. If the second argument is NULL, use the first
36 * as backing store. If the third argument is not NULL, it gets the time the
37 * snapshot was created. If the fourth argument is not NULL, it gets the
38 * snapshot device path.
39 */
40int snap_open(char *, char *, time_t *, char **);
